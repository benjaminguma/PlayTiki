# PlayTiki 

a music playlayer with audio visualization features

### Installing

```
npm install
npm run play-tiki
```
## Built With

- [React](https://reactjs.org/) - Reactjs
- [Express](https://expressjs.com/) - Express.js
- [MongoDb](https://www.mongodb.com/) - MongoDb
